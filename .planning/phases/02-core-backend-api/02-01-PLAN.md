## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-02-01 | Spoofing | /info, /download URL param | high | mitigate | URL validation regex (D-08) restricts to youtube.com/youtu.be domains only |
| T-02-02 | Tampering | format_id param | medium | mitigate | Validate format_id against available formats from yt-dlp; reject unknown IDs |
| T-02-03 | Repudiation | Download logging | low | accept | Single-user tool, no audit requirement |
| T-02-04 | Information Disclosure | Error messages | medium | mitigate | Generic error messages in production; detailed logs server-side only |
| T-02-05 | Denial of Service | Large downloads, rate limiting | high | mitigate | Streaming prevents memory exhaustion (D-04, D-05); temp file cleanup (D-07); yt-dlp rate limiting handled by upstream |
| T-02-06 | Elevation of Privilege | Path traversal via outtmpl | critical | mitigate | Hardcoded outtmpl template (D-09); tempfile.mkdtemp() isolation (D-06) |
| T-02-SC | Tampering | npm/pip/cargo installs | high | mitigate | slopcheck + blocking human checkpoint for [ASSUMED]/[SUS] packages |

</threat_model>
